# Todo-List-API
A simple Todo API built with FastAPI and SQLAlchemy
## Features
- FastAPI
- Create, Delete, Fetch Todos
- User accounts with password and authentication
## Tech Stack
- FastAPI
- SQLAlchemy ORM
- JWT Authentication
- Pwdlib
- Pydantic
## Database Schema
- User: manages authentication and ownership of todos
- Todo: stores tasks with title, description, and status
